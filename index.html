Restaurant Booking

Calendar-based restaurant reservation agent

  • Agent needs access to a calendar
  • Looks for events in the current week
  • If the event has an assigned place, it checks that it is a restaurant and can find its phone number on Google Places API
  • If there is a number, it generates a request for a booking depending on the number of people invited to the event
